# GuruGuru Sound

PositionalAudioを使って音像定位。

デモ https://vgurusnd.takosatou.io/

## 使い方

左のメニューの［ファイルを選択］をクリックして、音声ファイル（.mp3など）を開きます。
色に対応するボールのところから音が聞こえるように音像定位します。

音声ファイルはループ再生します。

最大4つまでの音を選べます。

音量

## 視点の移動方法
ボール以外の場所でマウスカーソルを左ボタンドラッグすると、視点を回転します。
右ボタンドラッグすると、視点を平行移動します。
マウスホイール操作で、視点を前後に移動します。

## ボールの移動方法
### インタラクティブ

ボールの上でドラッグすると、画面と平行な平面上でボールを移動します。

### 自動移動

横方向と縦（奥行）方向に、サインカーブ状に往復運動します。振幅と周期、位相を設定できます。

横と縦で周期が同じで、振幅が同じなら円、振幅異なれば楕円になります。

縦と横で周期が異なれば、リサージュ図形になります。

### プリセット

パン（往復運動）と円運動を簡単に設定できます。
